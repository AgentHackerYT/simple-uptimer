# Simple Uptimer
- Another Package from Agent's Lab which uptimes link you provide
# How to use (Functions)
```js
.uptime.create('link(must be http or https)')
```
```js
.uptime.interval('Number') // Optional
```
```js
.uptime.init()
// or
.uptime.init(result =>{
    console.log(result)
})
```
```js
.uptime.exit() // basically process.exit()
```
